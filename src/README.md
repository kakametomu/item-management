# 商品管理システム

## 概要
このシステムでは、店舗で扱う商品の在庫管理を行うことができます。

商品の新規登録、削除ができ、各商品の在庫を調整することができます。

## 主な機能
- ログイン、ログアウト機能
- 商品一覧画面
- 商品の新規登録、削除機能
- ユーザー一覧機能
- ユーザー削除機能
